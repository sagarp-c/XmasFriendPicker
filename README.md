# 🎄 Xmas Friend Picker

This is a simple **Python program** inspired by a real-life Christmas tradition. During Xmas, friends usually write their names on papers, shuffle them, and secretly pick one name. The picked friend stays **secret** until the day of gift giving.
This project recreates that same experience using code.

---

## 💡 What This Program Does

* Takes the names of all friends
* Shuffles them like folded papers
* Lets **each person choose a paper themselves**
* Ensures **no one gets their own name**
* Keeps everything **secret and encrypted**

To keep names hidden during the process, **Fernet encryption** is used, similar to how names are hidden inside folded papers in real life.

---

## ✨ Features

* Random and fair selection
* No self-picking
* Each friend chooses their own paper
* Names are encrypted for privacy
* Works as both a Python program and a web app

## 🔒 Why Encryption?

Just like folded papers hide names in real life, encryption hides names inside the program so:

* No one can see others’ choices
* The process stays fair and secret
